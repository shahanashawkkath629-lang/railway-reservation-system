🚆 Railway Reservation System

📌 Project Description

This is a Python-based railway reservation system that allows users to manage ticket bookings.

✨ Features

- Check seat availability
- Book tickets
- View ticket details
- Cancel tickets

🛠️ Technologies Used

- Python

▶️ How to Run

1. Download the project or clone repository
2. Run the following command:
   python main.py

🧪 Testing

- Supports multiple bookings
- Handles full capacity
- Allows ticket cancellation
